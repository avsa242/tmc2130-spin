# tmc2130-spin
---------------

This is a P8X32A/Propeller driver object for the Trinamic TMC2130 Stepper Motor driver IC.

## Salient Features

* SPI configuration (tested up to 1MHz)
* Chopper configuration: Short to GND Protection, Interpolation
